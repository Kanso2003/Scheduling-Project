# Scheduling Management System

Simple scheduling web app built with:

- Python Flask
- HTML / CSS / JavaScript
- Data stored in JSON files
- Login System with password hashing (SHA256)

## Features

- Admin Login
- CRUD Schedules (Create, Read, Update, Delete)
- JSON-based storage
- Flask framework

## Usage

```bash
To install all dependencies run: 
pip install -r requirements.txt

Run the app:
python app.py

admin login: user: admin ; password: admin123 (it's hashed)
